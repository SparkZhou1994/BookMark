# 语法解释
## あります / います
### 名(場所) に　名词(物) があります　/ 名词(人) がいます 	[xx有yy]
あそこ　に　本屋が　あります。
外　に　何が　ありますか。		(何后有助词读成なに，没有时读なん)
教室　に　学生が　います。
教室　に　だれが　いますか。
### 名词(物 / 人) は　名(場所) に　あります / います 	[yy在xx]
時計　は　どこ　にありますか。　＝　時計　は　どこ　ですか。
小野さん　は　事務所　にいます。
