# 语法
## 学习目标
名词修饰 
## 基本句型
### V简体 + N
N前不用敬体，です/ます　一般只出现在句尾。
V简体后 不加　の
修饰名词的小短句中主语用が

これは明日会議で使う資料です。
わたしが明日乗る飛行機は中国航空です。
中国で買ったCDを友達に貸しました。
### AN(现在肯定) な　+ N
AN的肯定现在/将来时需要加な，其他时态或形式直接用简体形
操作が簡単な　パソコンが欲しいです。
操作が簡単 ではない　パソコン
#### Aい(现在肯定)　+ N
入り口が広いあの建物がわしたの会社です。
入り口の広かった　建物
#### N(现在肯定) の　+ N
名词的肯定现在/将来时需要加の，其他时态或形式直接用简体形
数学が専門の先生が休みました。

ところ　所

でしたね。

真っ直ぐ　まっすぐ  [好直的]
真ん中　　まんなか  [正中的]
## あそこにある大きな白い建物は何ですか。
あそこにある，大きな，白い都是形容建物
如果只有大きい和白い可以写成
大きくて白い建物
